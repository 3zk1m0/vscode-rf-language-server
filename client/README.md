# Robot Framework Intellisense

A [Visual Studio Code](https://code.visualstudio.com/) extension that supports Robot Framework development.

## Features

### Syntax highlighting
* Supports `.robot` files

### Goto definition
* For variables
* For user keywords

### List file symbols
* Shows variables, keywords and test cases
